# arduino program sketch

## 1. download sketch
- <https://www.arduino.cc/en/Main/Software>
- 윈도우면 Windows Installer를 다운로드 한다.


## 2. sketch 프로그램
- 스케치 프로그램을 실행합니다. >> icon 클릭
- 그 다음 blink 예제를 켜봅니다.
	- 파일 -> 예제 -> 01.Basics -> Blink
- 아두이노를 USB 포트에 연결하고 다운로드를 합니다.
	- 툴 -> 포트
	- 체크 표시는 컴파일 
	- 화살표는 컴파일 후 다운로드
- 아두이노를 보면 13번 핀 옆에 LED가 깜빡이는 것을 볼 수 있다.
